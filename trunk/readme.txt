== Description ==

EZ Portfolio is easy to use customizable portfolio plugin for WordPress. Read documentation for usage.

== Usage ==

Read documentation located at /docs/ folder.

== Installation ==

Installing "EZ Portfolio" can be done either by searching for "EZ Portfolio" via the "Plugins > Add New" screen in your WordPress dashboard, or by using the following steps:

1. Download the plugin
1. Upload the ZIP file through the 'Plugins > Add New > Upload' screen in your WordPress dashboard
1. Activate the plugin through the 'Plugins' menu in WordPress
